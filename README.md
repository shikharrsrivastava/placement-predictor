# Placement Predictor

A Machine Learning web application that predicts whether a student is likely to be placed based on CGPA and IQ.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask

## Model Used

- Logistic Regression
- StandardScaler

## Files

- `mini_project.ipynb` - Training notebook
- `model.pkl` - Trained model
- `scaler.pkl` - Saved scaler
- `app.py` - Flask web app

## Run Locally

```bash
pip install flask scikit-learn numpy pandas
python app.py
```

Open:

http://127.0.0.1:5000
